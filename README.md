#  Delhi Metro Data Analytics Dashboard

An interactive data visualization dashboard built using **Python Libraries** to analyze Delhi Metro operational statistics.  
This project highlights key insights such as passenger trends, ticket revenue, fare variations, top stations performance, and overall network growth.

---

Dataset- https://www.kaggle.com/datasets/nikhilkumar766/delhi-metro-dataset
##  Features of the Dashboard

✔ Interactive filters for dates, stations, and ticket types  
✔ 6+ dynamic charts including:  
   - Line trend charts  
   - Bar charts  
   - Pie charts  
   - Scatter plots  
   - Comparative visuals  
✔ Responsive layout for web + desktop  
✔ Dataset-driven analytics with real-time updates  
✔ Clean and intuitive UI

---

##  Dataset Description

This dataset contains metro operations information collected from publicly available Delhi Metro sources.  
It includes:

| Feature | Description |
|--------|-------------|
| Date | Operational date |
| Station Name | Delhi Metro station |
| Passenger Count | Total number of passengers for the day |
| Ticket Revenue | Daily revenue from ticketing |
| Avg Distance | Average distance traveled per passenger |
| Ticket Type | Type of ticket issued (Smart Card, Token, etc.) |
| Top Stations Flag | Indicates top-performing stations |

>  NOTE: Data has been cleaned and processed for research and learning purposes.  
> This dataset does **not** contain personal or sensitive information.
